Manage your media and display them elegantly on your website.

== Description ==

This is the amazing PJS standard media manager built by the PlainJoe Studios team located in Southern California.

At a glance, this plugin does the following:

* Manages your media in a series like format.
* Displays your media elegantly to website visitors.
